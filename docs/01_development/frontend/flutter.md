## Introduction

Flutter est un framework open-source développé par Google pour créer des applications mobiles, web et de bureau à partir d'une seule base de code. Il utilise le langage de programmation Dart et offre une riche collection de widgets pour construire des interfaces utilisateur attrayantes et performantes.

## Création du projet

- Pour créer un nouveau projet Flutter, utilisez la commande suivante dans votre terminal

    ```bash
    flutter create nom_du_projet
    ```
